# 确实开源帮助中心

## 贡献文档方式
1.复刻主仓库

<br/>

2.进入复刻仓库编辑文档，可采用如下两种方式编辑：
* 克隆复刻仓库到本地后，在gitlink_help_center/docs文件夹下新加markdown文档，依次执行
```bash
 git add <新增文件>
 git commit <新增文件> -m "提交信息"
 git push
```
* 在gitlink代码仓库页面进行编辑，然后点击“提交变更”
![](https://gitlink.org.cn/api/attachments/412396)

<br/>

3.向主仓提交合并请求
