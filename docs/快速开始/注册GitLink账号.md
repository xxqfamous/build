# 注册GitLink账号

## 1. 点击**立即注册**按钮

![](/img/quickstart/register_homepage.png)

## 2. 填写注册信息

- 手机号注册

  ![](/img/quickstart/register_phone.png)

- 邮箱注册

  ![](/img/quickstart/register_email.png)

## 3. 注册完成

填写完所需信息后点击注册，注册成功后则进入个人主页

![](/img/quickstart/register_ownpage.png)
