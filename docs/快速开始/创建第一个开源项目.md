# 创建第一个开源项目

## 1. 新建项目

平台提供了“新建”按钮，用户可以通过点击快速从零开始创建新的公开或者私有项目。

![创建首页](/img/quickstart/create_homepage.png)

## 2. 填写项目信息

填写项目基本信息。

![](/img/quickstart/create_info.png)

## 3. 创建成功

点击创建项目，创建成功后进入项目主页。

![](/img/quickstart/create_success.png)
