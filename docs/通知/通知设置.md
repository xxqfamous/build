# 确实开源帮助中心

## 通知设置
#### 1.进入通知设置界面
通过点击首页的通知按钮进入通知设置界面；
![](/img/notice/home_notice.png)
<br/>
或在头像下拉列表中选择设置可以进入消息通知设置界面；
![](/img/notice/into_notice.png)

#### 2.进行通知设置
通过“通知管理”可以对接受通知的方式进行设置，默认所有通知都是通过站内信的方式接受，可以通过勾选为重要的通知类型增加邮件接受方式。

![](/img/notice/notice_set.png)
